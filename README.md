//-------------------------------3D Satellite Tracker-------------------------------//

A real-time 3D satellite visualization and tracking app built with Three.js, displaying satellites orbiting Earth with animated trails and city markers.

//-------------------------------Features-------------------------------//
3D Earth: Interactive globe with texture mapping and rotation.

Satellite Visualization: Add multiple satellites with RA/Dec positioning.

Orbit Trails: Animated trails follow satellites with a fading effect.

City Markers: Plot cities on the globe using latitude and longitude.

User Interaction:

OrbitControls for rotating and zooming.

GUI panel for camera adjustments.

Table-based UI to select and add satellites.

Responsive Design: Automatically adjusts to window resizing.

//-------------------------------Tech Stack-------------------------------//

Three.js – 3D rendering

HTML/CSS/JS – Frontend

Node.js + Express – Backend API for satellite data

//-------------------------------Notes-------------------------------//

Satellites are positioned using RA/Dec coordinates.

Trails dynamically follow satellites and fade over time.
